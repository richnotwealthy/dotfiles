# dotfiles

Portable dotfiles setup for new machines.

## What's Inside

| File | Purpose |
|------|---------|
| `zshrc.zsh` | Zsh configuration |
| `vimrc.vim` | Vim config for quick editing and navigation |
| `tmux.conf` | Tmux configuration |
| `theme.zsh` | Shell theme |
| `install` | Dotfiles installer |
| `install-tools` | Tool installer |
| `cloud-init.sh` | Cloud instance bootstrap |

## Setup

```sh
git clone <this-repo> ~/dotfiles
cd ~/dotfiles
./install
./install-tools
```
